
---

### ERROR[2-2-01]
#### 発生条件
- 匿名Ticketチャンネルを取得する際に、匿名Ticket送信チャンネルでの権限が不足している場合
#### 対処法
- サーバー管理者に`/pticket config`を実行してもらう
