### ERROR[3-4-01]
#### アプリケーションコマンド
- 【！サーバー管理者に報告】
#### 発生条件
- 匿名Report送信チャンネルが設定されていなかった場合
- jsonファイルが存在しなかった場合
#### 対処法
- サーバー管理者に`/settings`を実行してもらう

### ERROR[3-4-02]
#### アプリケーションコマンド
- 【！サーバー管理者に報告】
#### 発生条件
- 匿名Report送信チャンネルが設定されていなかった場合

### ERROR[3-4-03]
#### モーダル

#### 対処法
