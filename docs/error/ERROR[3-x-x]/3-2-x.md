### ERROR[3-2-01]
#### コマンド
- ユーザーがDMにてサーバー管理者からのメッセージに返信する
#### 発生条件
- 匿名Reportチャンネルを取得する際に、権限が不足していることが分かった場合
#### 対処法
- サーバー管理者に`/report config`を実行してもらう
